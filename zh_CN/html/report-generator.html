Description

The report generator allows to combine multiple statistics for sending/downloading. Reports can be configured to be sent automatically on specified times or run manually.

Configuration

No further configuration needed.

Usage

Creating Reports

After you've entered title and description for your report in the first step, you'll be able to set up your report.

General Settings

Set up title, description and validity of the report. This information will not be visible in the report.

Figure 3.25. Automatic generation settings


Automatic Generation Settings

If the report should be generated automatically (e.g. once per day), you can set up subject, text and recipients for the email which is going to be sent and which will contain the report as an attachment. You'll also need to add information on how often the report should be sent. The data format is taken from the cron mechanism of unix-based operating systems. For more information on which settings are possible, please refer to official cron documentation sources or wikipedia.

Figure 3.26. Automatic generation settings


Output Settings

In this section, you can set up additional information which should be contained in the report (e.g. a preamble or epilogue). This could e.g. be used for adding copyright or data protection information.
